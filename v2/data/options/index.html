Other Options

Path to the executable2:
This extension supports managed storage. Some of the preferences can be pre-configured by the domain administrator
To apply command-line options, create a batch file, and insert the path of this batch file inside the "Path to the executable".

-

  1. It is possible to use "managed storage" to set the default host and URL list. This is useful for administrators to force opening a set of URLs or hostnames in another browser. Instruction on how to setup the managed storage can be found in the FAQs page.
  2. On Windows, if the path to the executable is not provided (is empty) %ProgramFiles%\\Opera\\launcher.exe is used. On Linux, if the path is not set, it is assumed opera binary can be found in the global PATH environment. On Mac OS, open -a command is used (with opera as the application or whatever is in the input).
  3. By activating this option all the requested URLs are sent to the Opera browser at once. Only activate this option if the Opera browser in your operating system is capable of handling this type of requests.
  4. Not all key combinations are allowed in all operating systems.
  5. When enabled, all left-click links except the ones that match with at least one condition will be sent to the Opera browser.
  6. If checked, the extension validates top-level navigation with the list of left-click hostnames and URLs. If matched, the URL is sent to the Opera browser, and the navigation is blocked.
